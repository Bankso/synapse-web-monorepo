  • description: This is version 1.2 of the data release (Production Date: 2024-05-10), described as the 0.5 alpha data release of the Cell Maps for Artificial Intelligence (CM4AI) Functional Genomics Data Generation Project, a component of the U.S. National Institute of Health’s (NIH) Bridge2AI program. CM4AI’s objective is to deliver machine-readable hierarchical maps of cell architecture as AI-Ready data produced from multimodal interrogation of 100 chromatin modifiers and 100 metabolic enzymes involved in cancer, neuropsychiatric, and cardiac disorders in disease-relevant cell lines under perturbed and unperturbed conditions. Input data streams are integrated via the multi-scale integrated cell (MuSIC) software pipeline and output cell maps are packaged with provenance graphs and rich metadata as AI-Ready datasets in RO-Crate format using an extended, client-server version of the FAIRSCAPE framework.                                                                     • purposes:
                                                                      • CM4AI’s objective is to deliver machine-readable hierarchical maps of cell architecture as AI-Ready data produced from multimodal interrogation of 100 chromatin modifiers and 100 metabolic enzymes involved in cancer, neuropsychiatric, and cardiac disorders in disease-relevant cell lines under perturbed and unperturbed conditions, utilizing state-of-the-art mass spectrometry based proteomics, spatial proteomics / cell imaging, and genetic perturbations using CRISPR.
                                                                    • tasks:
                                                                      • To enable AI/ML research on cell architecture, particularly related to cancer, neuropsychiatric, and cardiac disorders by providing integrated, machine-readable cell maps from multimodal interrogation of cellular components.
                                                                    • instances:
                                                                      • Instance Type: Hierarchical cell maps, immunofluorescence (IF) subcellular microscopy images, affinity purification mass spectroscopy (AP-MS) data, size exclusion mass spectroscopy (SEC-MS) data, single-cell CRISPR-Cas perturbation screen data.
                                                                        • description: The dataset comprises AI-Ready data including spatial proteomics (IF microscopy), protein-protein interaction data (AP-MS, SEC-MS), and genetic perturbation screen results from disease-relevant cell lines. These are integrated into hierarchical maps of cell architecture.
                                                                    • acquisition_methods:
                                                                      • Data was generated from multimodal interrogation of 100 chromatin modifiers and 100 metabolic enzymes in disease-relevant cell lines, under both perturbed and unperturbed conditions. Specific methods include immunofluorescence (IF) subcellular microscopy for spatial proteomics data; affinity purification mass spectroscopy (AP-MS) and size exclusion mass spectroscopy (SEC-MS) for protein-protein interaction (PPI) data; and single-cell CRISPR-Cas perturbation screens by cell type.
                                                                    • collection_mechanisms:
                                                                      • The data generation utilized state-of-the-art mass spectrometry based proteomics, spatial proteomics / cell imaging techniques, and genetic perturbations using CRISPR technology.
                                                                    • data_collectors:
                                                                      • The Cell Maps for Artificial Intelligence (CM4AI) project team and collaborators, as listed in the dataset authorship.
                                                                    • preprocessing_strategies:
                                                                        • description: Input data streams are integrated via the multi-scale integrated cell (MuSIC) software pipeline, which employs deep learning models and community detection algorithms. The output cell maps are packaged with provenance graphs and rich metadata as AI-Ready datasets in RO-Crate format using an extended, client-server version of the FAIRSCAPE framework.
